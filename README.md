## Used Technologies
**Back-end:** Node.js -> bcrypt, express, jsonwebtoken, mongoose, winston

**Front-end:** React -> tailwindcss, axios

**Data-base:** MongoDb

## Run
Clone the project

```bash
  git clone https://github.com/dilsad-erdogan/fitness-tracker.git
```

- Backend

Go to project directory

```bash
  cd backend_nodejs
```

Run project

```bash
  npm start
```

- Frontend

Go to project directory

```bash
  cd client
```

Run project

```bash
  npm start
```

## Environment Variables
To run this project you will need to add the .env file in the 'backend_nodejs' folder.

`PORT`
`JWT_SECRET`
`JWT_EXPIRATION`
`MONGODB_URI`
`SMTP_HOST`
`SMTP_PORT`
`SMTP_USER`
`SMTP_PASS`
`FROM_NAME`
`FROM_EMAIL`
